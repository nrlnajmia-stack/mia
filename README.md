# Portfolio Web

Proyek portofolio single-page dengan PHP native, Tailwind CSS, dan struktur data terpusat.

## Struktur
- `index.php` — halaman utama
- `contact-process.php` — pemroses formulir kontak
- `includes/` — potongan template section
- `data/` — konten `skills`, `projects`, `case-studies`, `organizations`
- `assets/` — CSS, JavaScript, ikon, dan gambar placeholder
- `lib/validate.php` — fungsi sanitasi dan validasi input

## Cara menjalankan
1. Pastikan server PHP berjalan.
2. Letakkan folder `portofolio` di direktori server web Anda.
3. Buka `http://localhost/path-ke-portofolio/index.php`.

## Catatan
- Ganti alamat email tujuan di `contact-process.php` dengan alamat pemilik.
- Ganti placeholder gambar di `assets/images/` dengan aset portofolio nyata.
